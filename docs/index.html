Features

Everything you need.
Nothing you don't.

Professional disk cleaning without the bloat or the price tag.

30-Second Scan

Full system scan in under 30 seconds. See exactly where your space is going with a visual breakdown.

Safe by Design

System-critical paths are blocklisted. Files go to Trash by default. Confirmation before every action.

12 Categories

Caches, logs, Xcode data, Docker, node_modules, iOS backups, large files, and more. All in one place.

File Inspector

Browse every file per category. Select individual items, sort by size, and review before deleting.

Fully Configurable

Toggle categories, set file age thresholds, choose between Trash or permanent delete. Your rules.

Native macOS UI

Built with SwiftUI. Sidebar navigation, SF Symbols, Swift Charts, dark mode. Feels like it belongs.
